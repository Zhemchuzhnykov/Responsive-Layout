# Responsive Layout: mobile - desktop versions.

---

The breakpoint is **360px**: it changes at 360px from the mobile to the desktop version which remains the same at any further resolution.

## The tech stack is:

- HTML5
- CSS3
- Flexbox
- Sass (Scss)
- BEM methodology
